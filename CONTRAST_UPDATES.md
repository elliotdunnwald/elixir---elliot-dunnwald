# Contrast & Visibility Updates

## Text Color Changes

All grey text has been updated to white/off-white for better readability:

- **text-zinc-500** → **text-zinc-100** (light off-white)
- **text-zinc-600** → **text-zinc-200** (lighter off-white)

## Selection States

All clickable elements now have clear white borders to indicate:
1. **Hover state**: Border changes from zinc-800 → zinc-600
2. **Active/Selected state**: White border + optional white background glow
3. **Focus state**: White border highlight

## Components Updated

### Navigation
- Desktop nav links: zinc-100 text, white when active
- Mobile nav icons: White border box when selected
- Buttons: Consistent bordered style with hover effects

### PostCard (Feed Items)
- Action buttons (Like, Comment, Share): Bordered pills, white border when active
- User info: Off-white text (zinc-100) instead of grey
- Timestamps: Light grey (zinc-200)
- Parameter labels: Light grey (zinc-200)

### Profile Page
- Section labels: Off-white (zinc-100)
- Location text: Off-white (zinc-100)
- Stats labels: Light grey (zinc-200)
- Tab buttons: White border + background when selected
- Action buttons: Clear borders with hover states

### Forms & Inputs
- All form labels: Off-white (zinc-100)
- Placeholder text: Remains subtle (zinc-700-800)
- Toggle buttons: White when selected
- Search inputs: Icon changes to white on focus

### Modals
- Edit Profile: All labels updated to off-white
- Brew Log Modal: All labels updated to off-white
- Image Crop Modal: All text updated for visibility

## Design Principles

1. **Readability First**: All readable text is now white or off-white
2. **Clear Selection**: White borders indicate active/selected states
3. **Consistent Hierarchy**:
   - Primary text: White
   - Secondary labels: zinc-100 (light off-white)
   - Tertiary info: zinc-200 (lighter grey)
   - Subtle elements: zinc-700+ (very subtle)
4. **Hover Feedback**: All interactive elements have visible hover states

## Color Palette

- **White** (#ffffff): Primary buttons, selected states, primary text
- **zinc-100** (#f4f4f5): Labels, secondary text
- **zinc-200** (#e4e4e7): Timestamps, metadata
- **zinc-700** (#3f3f46): Subtle icons, placeholders
- **zinc-800** (#27272a): Default borders, backgrounds
- **zinc-900** (#18181b): Card backgrounds
- **Black** (#000000): Main background

All changes maintain the black/white aesthetic while dramatically improving usability and accessibility.
